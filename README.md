# ☁️ Azure Project – My-Azure-project-

## 📌 Project Overview
This project demonstrates the deployment and management of cloud resources using Microsoft Azure. It showcases my understanding of Azure services, cloud infrastructure, and basic deployment processes.

## 🎯 Objectives
- To deploy cloud resources on Microsoft Azure
- To understand Azure resource configuration
- To demonstrate practical cloud computing skills

## ☁️ Azure Services Used
- Azure Virtual Machine (VM)
- Azure Resource Group
- Azure Storage Account
- Azure Portal

## ⚙️ Project Implementation
Account Creation (Azure Free Tier)

## Step-by-Step Signup Process
1. Visit: https://azure.microsoft.com/free  
2. Click **Start Free**
3. Sign in or create a Microsoft account  
4. Enter personal details (Name, Country, Email)

## 🔐 Verification प्रक्रिया
- **Email Verification** → Enter code sent to email  
- **Phone Verification** → Enter OTP sent to phone  
- **Credit Card Verification** → Required for identity (not charged)

## 🎁 Free Tier Benefits
- $200 credit for 30 days  
- Free access to selected services for 12 months

# 🔐 6. Security (MFA Setup)

## Enable Multi-Factor Authentication

### Steps:
1. Go to **Azure Active Directory**
2. Click **Users**
3. Select your account
4. Enable **Multi-Factor Authentication**
5. Configure:
   - Authenticator App OR
   - SMS verification

---
